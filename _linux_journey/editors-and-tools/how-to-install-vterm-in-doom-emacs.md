# How to Install vterm in Doom Emacs

* Enable `vterm` under `init.el`

* SPC h r r

* Install `libvterm` with `dnf install -y libvterm`

* Install `cmake` with `dnf install -y cmake`

* Run `vterm` with `M-x vterm`
