# How to Make KDE Look Like Windows 10 - for KDE6.x

* KWin Scripts --> "Get New Scripts" --> install "Sticky Window Snapping" (latest version).

    * Enable the script and click "Apply"
    
* Screen Edges --> set to "No Action"

* Global Theme --> "Get New..." --> search and install "We10XOS-dark"

* Ignore any errors or warnings presented.

* Go back to "Global Theme" and select the them. Enable "Appearance settings" and "Desktop and window layout" and click "Apply"

* Go to "Icons" and select "We10X-black-dark" and then click "Apply"

* Go to "Desktop Effects" and enabled "Translucency"

* Set "Inactive windows" and "Moving windows" three notches up from "Opaque".

Finished.
