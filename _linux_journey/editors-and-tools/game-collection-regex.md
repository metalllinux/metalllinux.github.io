 * `/200./g`
	* Searches for all games made from 2000 ~ 2009.
* `/PS[2-3]/g`
	* Searches for only PS2 and PS3 games.
* `/Another one to beat. \w*/`
	* Searches for games I would like to beat and if there was anything else interesting of note afterwards in the comments.
* `\d{4}\/10\/\d{2}`
	* Finds any games that were released in October (good for Halloween).
