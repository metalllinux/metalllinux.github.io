---
title: "How Set Up Go(Lang) With Neo Vim"
category: "editors-and-tools"
tags: ["editors-and-tools", "golang", "neo", "vim"]
---

Navigate to https://vim-bootstrap.com/#tagline and create the generate.vim file.
Move that file to `~/.config/nvim/init.vim` (you may need to create the `nvim` directory).
Open `nvim` and it will download the packages automatically.
Make sure to run `:GoInstallBinaries`
