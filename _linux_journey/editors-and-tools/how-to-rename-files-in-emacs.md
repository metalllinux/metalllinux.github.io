# How to Rename Files in Emacs

* Open `dired`:
```
C + x, then d
```

* Press "enter" on the directory where the file is located.

* Search for the file.

* Press `R` and then rename the file.

Else from the Buffer:

```
M-x
```

* Select `rename-file`
