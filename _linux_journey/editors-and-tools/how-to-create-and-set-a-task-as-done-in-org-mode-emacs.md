# How to Create a Task in Org Mode in Emacs

* Create a task with the `TODO` command, like so:
```
* TODO Weekly review
  SCHEDULED: <2025-06-06 Fri +1w>
```

* To set a task as `DONE`, either change `TODO` to `DONE` or use the keyboard shortcut `C-c C-t` while the cursor is on the task itself.
