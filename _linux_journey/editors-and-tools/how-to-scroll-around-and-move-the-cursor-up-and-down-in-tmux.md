# How to Scroll Around and Move the Cursor Up and Down in tmux

```
Ctrl + b then [
```

```
Ctrl + u to scroll up by half a screen

Ctrl + d to scroll down by half a screen
```

```
Press q or Enter to exit copy mode.
```
