# How Set a Flatpak to Dark Mode with Flatseal 

* Go to the application you want to set to Dark Mode and scroll down to `Variables`.

* Add the following for GNOME / GTK:

```
GTK_THEME=Adwaita:dark
```


* Restart the application.
