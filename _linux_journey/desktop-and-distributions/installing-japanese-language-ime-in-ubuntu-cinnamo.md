---
title: "Installing Japanese Language Ime In Ubuntu Cinnamo"
category: "desktop-and-distributions"
tags: ["desktop-and-distributions", "installing", "japanese", "language", "ime"]
---

* Go to Language Support --> select Japanese and install it.
* Log out and then back in again.
* Go to Preferences --> Add Keyboard --> Mozc and then you have access to the Japanes IME.
