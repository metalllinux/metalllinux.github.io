---
title: "If You Need To Add An External Library To Immich"
category: "general-linux"
tags: ["you", "need", "add", "external", "library"]
---

* Follow the instructions here https://immich.app/docs/features/libraries/ from **Mount Docker Volumes**:
* If you have any running containers, make sure you stop them with `docker stop <container_name>`/
* Once you have the directories added to the docker-compose.yml file, then run `docker compose up` to initiate the changes.
