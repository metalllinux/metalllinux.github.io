title: "Install On Blendos Host System"
category: "general-linux"
tags: ["install", "blendos", "host", "system"]
---

Install a package
Run system install [pkg] to install a package on the host from the Arch Linux repositories.

Remove a package
Run system remove [pkg] to remove a package from the host.

Switching tracks
Run system track to be presented with a list of desktop environment options you can switch to, and select any one.
