---
title: "How Access Proton Directories"
category: "general-linux"
tags: ["access", "proton", "directories"]
---

To access a proton's file system directory in Steam, you need to follow these steps:

    Open Steam and go to your Library.
    Right-click on the game you want to access the files for and select "Properties".
    In the Properties window, go to the "Local Files" tab and click on "Browse Local Files".
    This will open the game's directory on your computer.
    Within this directory, you should be able to access the files related to the game's installation and configuration.

Note that if you are running the game through Proton, the directory you access may be located in the Proton prefix folder rather than the game's installation folder. The Proton prefix folder can be found in the Steam installation directory under "steam/steamapps/compatdata". Look for the folder with the same numerical ID as the game you are trying to access.
