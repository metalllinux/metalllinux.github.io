---
title: "Creating A Soft Symbolic Link"
category: "general-linux"
tags: ["creating", "soft", "symbolic", "link"]
---

```
ln -s <path to the file/folder to be linked> <the path of the link to be created>
```
* Example includes:
```
ln -s /usr/bin/nano ~/testing/2024-04-26/ciqctl_alternatives_nano_symlink
```
