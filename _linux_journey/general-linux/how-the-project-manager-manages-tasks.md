# How the Project Manager Manages Tasks

* Project Managers start their day looking at the detailed project plan.

    * Ask themselves - are things on track and are tasks ready to start.
    
    * Have to look at the Project Diary and ask owners if they are on target with their committments.
    
* The Project Manager has to manage the work AND people.

* Critical Path is key - the Project Manager has to make sure this is not delayed.

* Productivity is the ratio between output and input.

* Good productivity is when you have high output and low input.

* Time and Cost push towards Scope.

    * Software design - the number of functionalities programmed. 

* Project Managers are on constant look-out on optimising the workflow.

    * Constantly ask if there is a better way to sequence the work - get feedback from the team members.
    
* Don't forget other activities aside from the Critical Path. If these activities get delayed, these could then be the Critical Path.

* The other parts that require management are PEOPLE.
