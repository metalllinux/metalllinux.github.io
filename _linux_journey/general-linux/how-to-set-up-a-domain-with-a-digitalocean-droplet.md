---
title: "How To Set Up A Domain With A Digitalocean Droplet"
category: "general-linux"
tags: ["domain", "digitalocean", "droplet"]
---

* Set up Namecheap and point the Nameservers to DigitalOcean.
* In the DigitalOcean console, go to Networking and then add the Droplet IP with the domain name and an @ to make it an A record.
