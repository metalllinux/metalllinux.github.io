# What Does the Rocky Linux Storage SIG do?

Currently there is an on-going effort to get ceph-reef for Rocky 8 and ceph-squid and ceph-tentacle for Rocky 9 and 10. Ostensibly there will be some kind of ceph release package but first steps are to get Peridot projects going and then build the release packages.
