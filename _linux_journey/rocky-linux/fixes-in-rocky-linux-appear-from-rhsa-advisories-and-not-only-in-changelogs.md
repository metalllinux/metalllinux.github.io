# Fixes in Rocky Linux Can Come in the Form of Multiple Advisory Fixes and Not Only in the Changelogs

These CVE's may not be outlined in a changelog but were probably fixed as part of multiple advisory fixes. If they're marked as fixed in any older RHEL release, they're also fixed in any Rocky versions
