# How to Install Rocky Linux on the Liva Z Mini PC

* #Rocky Linux 10 just hangs, this does not work when booting via USB.

* Rocky Linux 9.x works.

* Need to set up a USB stick with Ventoy. Then the machine will boot.

* Set the BIOS to boot a Windows 7.x or under device and also only select the USB stick as the boot device and disable all other devices.
