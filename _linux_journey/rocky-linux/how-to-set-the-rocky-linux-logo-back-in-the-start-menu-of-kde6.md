# How to Set the Rocky Linux Logo Back in the Start Menu of KDE

* Right click the application launcher and go to "Configure Application Launcher".

* Click the "Icon" and then "Choose".

* Then in the drop-down menu, hit "Other" and you'll see the Rocky Linux logo.
