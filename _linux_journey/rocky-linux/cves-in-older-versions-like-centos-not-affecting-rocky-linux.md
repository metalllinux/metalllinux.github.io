# CVEs in Older Versions Like CentOS Not Affecting Rocky Linux

By virtue of the CVE only affected an old version not in rocky and rocky having a newer version you can consider these not relevant

The vast majority of CVEs listed apply to very old OpenSSL versions tied to RHEL 2, 3, 4, 5, 6, or 7 .
Where RHEL 7 is mentioned, the advisories are always against OpenSSL 1.0.1/1.0.2 or older branches, which are not present in Rocky 8.
No CVEs in your file reference RHEL 8 as an affected operating system.
