# Kernel receives unexpected unknown NMI in Red Hat Enterprise Linux 9.3

Solution Unverified \- Updated November 15 2024 at 2:07 AM -

<a id="dLabel"></a>English

## Environment

- Red Hat Enterprise Linux 9.3

## Issue

- Sometimes the kernel shows the following message without NMI button operations.
    
    [Raw](#)
    
    ```
    kernel: Uhhuh. NMI received for unknown reason 2c on CPU 0.
    ```
    

## Resolution

- Update to `kernel-5.14.0-503.11.1.el9_5` or later

## Root Cause

- Kernel patch(s) introduced in Red Hat Enterprise Linux 9.3 could cause the issue.
