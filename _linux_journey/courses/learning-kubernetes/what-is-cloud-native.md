* Kubernetes is a Cloud Native technology.
* Cloud Natives technologies allow empower organisations to build and run scalable applications in public, private and hybrid clouds. Containers, service meshes, microservices, immutable infrastructure and declarative APIs exemplify the approach. - CNCF.org - Cloud Native Computing Foundation
* Another definition is --> Cloud Native technologies are open-source projects designed to allow technologists use cloud computing services to automatically deploy and scale applications.
* Cloud Native technology is built in mind to remove barriers between System Administrators --> Infrastructure Engineers --> Software Engineers.
