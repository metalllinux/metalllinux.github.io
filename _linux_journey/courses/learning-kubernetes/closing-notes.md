* To keep up with the project, check for announcements from cncf.io
* To go deeper into Kubernetes, there is the path `Master Cloud-Native Infrastructure with Kubernetes` on LinkedIn Learning.
* Another good way to learn, is from talks presented at KubeCon.
* A good way to practice on a cluster, is taking one of the Linux Foundation Kubernetes Certificate Exams.
	* Kubernetes and Cloud Native Associate Exam.
	* Certified Kubernetes Application Developer Exam.
	* Certified Kubernetes Administrator Exam.
