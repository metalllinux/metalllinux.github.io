* Back button - `modify(Alt_L, key(left))`
* Left Skip button - `KEY_LEFT`
* Right Skip button - `KEY_RIGHT`
* Play button - `KEY_SPACE`
* i button - `f`
* Down button - `mouse(down,3)`
* Left button - `mouse(left,3)`
* Centre Enter button - `key(BTN_LEFT)`
* Right button - `mouse(right,3)`
* Up button - `mouse(up,3)`
