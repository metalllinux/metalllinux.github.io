```
sudo apt install -y libdvd-pkg
```
* Then run:
```
sudo dpkg-reconfigure libdvd-pkg
```
* If need to update the package, run the below command:
```
sudo dpkg-reconfigure libdvd-pkg
```
* Install VLC Player:
```
sudo apt install -y vlc
```
* Set VLC to Autostart in the KDE settings.
